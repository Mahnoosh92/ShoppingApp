# ShoppingApp
An app for shopping with detail and profile pages
<br />
<br />
# Technologies & Methodologies which used:
- MVI Pattern
- Clean-architecture
- Koin
- LiveData & Flow
- Coil(Image Loader)
- Room & Retrofit
- Unit testing
- Turbine for flow testing
- Espresso
